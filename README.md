1. **Clone the repository:**
     ```bash

        cd Blogify-website
     ```

     ```bash
         npm install
     ```
4. **Create a .env file for environment variables**:  
    - Add the necessary variables (e.g., database URI, Cloudinary credentials, etc).
5. **Run the project:**
     ```bash
         npm start
     ```
